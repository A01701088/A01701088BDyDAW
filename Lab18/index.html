Preguntas

¿Qué importancia tiene AJAX en el desarrollo de RIA's (Rich Internet Applications?

Poder hacer requests al servidor sin tener que recargar la página permite crear aplicaciones web de alto nivel que son agradables para el usuario.

¿Qué implicaciones de seguridad tiene AJAX? ¿Dónde se deben hacer las validaciones de seguridad, del lado del cliente o del lado del servidor?

AJAX manda solicitudes HTTP y por lo tanto se deben de hacer validaciones en el servidor y cliente ya que no puedes dejar que un cliente mande solicitudes al servidor sin verificarlos.

¿Qué es JSON?

Es Javascript Object Notation. Es una manera de transportar informacion del servidor al cliente y que es legible y ligero.
