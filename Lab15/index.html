Preguntas

¿Por qué es una buena práctica separar el modelo del controlador?

Es buena idea separar el model del controlador para tener mejor mantenibilidad si tienes que cambiar algo relacionado a la base de datos.

¿Qué es SQL injection y cómo se puede prevenir?

SQL injection es inyectar consultas de SQL a una forma y se puede prevenir con mysqli_prepare que prepara la consulta para que no puedan meter otra consulta complete en el string.
