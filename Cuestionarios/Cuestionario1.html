Cuestionario 1

¿Qué es entonces una base de datos - DB?

Una base de datos es un almacén de datos en un sistema digital. Estos datos están organizados para poder acceder a ellos.

¿En qué casos es conveniente usar bases de datos?

Se usa una base de datos cuando un pedazo de información se tiene que guardar y organizar.

¿Qué es un sistema de gestión de base de datos - DBMS?

Un sistema de gestión es como un usuario accede a la base de datos. Una base de datos es básicamente ilegible para un humano cuando llega a un cierto tamaño y por eso se necesita una interfaz entre la BD y la persona.

Enlista y explica las funciones/responsabilidades que tienen los DBMS.

- Seguridad

- Manejar los datos

- Simplicidad y uso fácil

- Abstracción

Ejemplifica y justifica en que proyecto de los que has realizado hubiera sido conveniente utilizar una base de datos.

En un proyecto donde tuvimos que hacer un sistema de cajeros automáticos hubiera sido más fácil crear una base de datos en vez de usar miles de apuntadores entre los varios objetos que creamos en C++.
